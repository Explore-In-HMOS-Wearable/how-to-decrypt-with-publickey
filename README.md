# How to decrypt with public key

This project demonstrates the how to decrypt with public key on Huawei smart watches.

# Use Cases

-This project shows how to use public key for the decrypt in RSA|ECB|PKCS1Padding algorithm.

**Frameworks**: HarmonyOS SDK 5.1.0
**Tools**: DevEco Studio 5.1.0
**Libraries/Kits**:
- @kit.ArkTS
- @kit.CryptoArchitectureKit
